    Учим пор­ту­галь­ский

    Боль­шин­ство ре­сур­сов на ан­глий­ском

    * falar.ru — огром­ные за­ле­жи ин­фор­ма­ции по пор­ту­галь­ско­му, по­жа­луй, ос­нов­ное ме­сто для изу­че­ния (на рус­ском!)

    * Он-лайн пе­ре­вод­чик: online.multilex.ru
    * хо­ро­ший пор­ту­галь­ский тол­ко­вый сло­варь: priberam.pt (есть по­ме­та Bras. для бра­зиль­ско­го ва­ри­ан­та :) все фор­мы гла­го­ла то­же мож­но по­смот­реть, на­жав ссыл­ку «conjugar») (via Lua)
     * тол­ко­вый сло­варь бра­зиль­ской ненор­ма­тив­ной лек­си­ки: dicionarioinformal.com.br
    * сло­варь пор­ту­галь­ско­го жар­го­на: пе­ре­вод на ан­глий­ском, пе­ре­вод mOrigammi
    * Ин­декс по пор­ту­галь­ской грам­ма­ти­ке: http://www.deltatranslator.com/delta/grammar_index.htm
    * Все фор­мы гла­го­лов по ин­фи­ни­ти­ву: http://linguistica.insite.com.br/cgi-bin/conjugue

    * Учим пор­ту­галь­ский: http://br.geocities.com/konkero/curso.htm
    * класс­ный учеб­ник раз­го­вор­но­го бра­зиль­ско­го http://biblioverde.narod.ru/lang/portugal/colloquail_portuguese.html
    * „Fala Brasil“ — Teimoso version :) http://narod.ru/disk/1236811000/Fala_Brasil.pdf.html
    * учеб­ник Allen Portugal (на рус­ском) http://narod.ru/disk/7499707000/Allen_Portugal_uchebnik.zip.html
    * Пор­ту­галь­ский для ка­по­э­ри­стов (!) на англ. http://www.capsprings.com/Portuguese/Introduction.aspx

    * Про­ект Ильи Фран­ка: http://www.franklang.ru/index.php?option=com_content&task=category§ionid=2&id=39&Itemid=39
    * Еще один муль­ти­я­зы­ко­вой про­ект, мно­го ре­сур­сов для ска­чи­ва­ния: uztranslations.net.ru (via GPS)
     * Ка­то Ломб «Как я изу­чаю язы­ки» http://www.gumer.info/bibliotek_Buks/Linguist/lomb/index.php
