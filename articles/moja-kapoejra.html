    26.10.2014 Новости и отчеты     накопилось, капоэйра-ангола-мать-зовет-домой, "вадиар-слово-о-свободе", скучно пиздец сил нет

    моя ка­по­эй­ра

    из со­про­тив­ле­ния, из взгля­дов де­тей и ста­ри­ков из-за уг­лов об­стре­лян­ных са­ра­ев в тру­що­бах го­ро­дов, из ду­ха угне­тен­но­го на­ро­да, дис­кри­ми­на­ция ко­то­ро­го про­дол­жа­ет­ся и по сей день ру­ка­ми мест­ных псов- жан­дар­мов, во­ору­жен­ных fazendeiros на план­та­ци­ях и бле­стя­щих вы­ве­сок кор­по­ра­ций, ве­ду­щих че­ло­ве­че­ство еди­ной до­ро­гой в свет­лое бу­ду­щее.

    вне си­сте­мы, без фла­га, без ло­го­ти­па,раз­де­ле­ний, псев­до­фи­ло­со­фий с счаст­ли­вы­ми и вы­дав­лен­ны­ми улыб­ка­ми иди­о­тов на­по­до­бие "гар­мо­нии/ми­ра/люб­ви/ра­до­сти иг­ры и про­чих фаль­ши­вых ро­зо­вых соп­лей ли­бо та­ких же при­дур­ко­ва­тых и се­рьез­ных из се­рии «ве­ли­кое бо­е­вое ис­кус­ство/сол­да­ты мест­ре бим­бы», без под­ме­ны по­ня­тий, без уни­фор­мы, без ро­бо­ти­зи­ро­ван­ных кло­нов, без ав­то­ри­та­риз­ма и при­над­леж­но­стей.

    без групп, без школ, без ти­ту­лов, без гра­ду­а­ции, без взи­ма­ния пла­ты, без сти­лей, без со­рев­но­ва­ний, без бе­ся­чей на­пы­щен­ной идео­ло­гии, по­пугай­ских по­вто­ре­ний штам­пов и бес­ко­неч­ных ко­пи­па­с­тов на те­му двух „ос­нов­ных“ пу­тей и ли­ний «бим­ба — ре­жио­нал, паш­ти­нья — Ка­по­эй­ра-Мать-Ан­го­ла чер­но-жел­тое брат­ство»

    из виб­ра­ции ка­ба­сы как зо­ва язы­че­ско­го ле­са мо­их пред­ков, из кро­ви ты­сяч за­губ­лен­ных за­пад­ны­ми вар­ва­ра­ми ин­дей­цев и аф­ри­кан­цев, из ко­жи ба­ра­ба­на, из за­ро­с­лей, из острия ма­че­те и ко­пья, из оран­же­во­го за­ка­та на го­ри­зон­те и шу­ма волн.

    из есте­ствен­но­сти, из про­сто­ты, из на­ту­раль­но­сти, из по­пущен­но­сти, из де­рев­ни, из фа­ве­лы, из ули­цы, из спон­тан­ной ро­ды, из рва­ных шта­нов, гряз­ных фут­бо­лок и фин­га­лов под гла­зом на до­воль­ных физио­но­ми­ях, из несколь­ких че­ло­век в ба­те­рии да дво­их иг­ра­ю­щих.

    без лос­ка, без мас­со­во­сти, без по­ка­за­тель­но­сти, без до­ка­зы­ва­ния, без кон­ку­рен­ции, без по­сто­ян­но­го и бес­по­кой­но­го стрем­ле­ния за­нять ни­шу и рас­ши­рить гео­гра­фию в пре­де­лах ка­мен­ных джун­глей.

    моя ка­по­эй­ра.
